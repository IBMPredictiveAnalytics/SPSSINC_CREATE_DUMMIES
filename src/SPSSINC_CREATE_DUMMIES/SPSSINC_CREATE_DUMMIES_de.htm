Dummy-Variablenset aus Werten einer Variablen erstellen

Bei dieser Prozedur wird aus einer Liste von Variablen ein Set von Dummy-Variablen erstellt, die die zugehörigen Werte darstellen. Dabei können Dummys für einzelne Variablen sowie Zweiwege- und/oder Dreiwegeinteraktionsterme erstellt werden. Die Prozedur gibt die Daten weiter, wobei alle durch den aktuellen Filter ausgeschlossenen Fälle ignoriert werden, und erstellt Dummys für die gefundenen Werte.

Bei metrischen Variablen erstellt sie standardmäßig keine Dummys, sondern nur eine Variable, die zum Erstellen von Interaktionstermen verwendet werden kann.

Dummy-Variablen erstellen für. Wählen Sie Variablen aus, aus denen Sie Sets von Dummy-Variablen erstellen. In der Regel handelt es sich dabei um kategoriale Variablen. Aber es ist sinnvoll, metrische Variablen zum Erstellen von Interaktionstermen auszuwählen. Es können sowohl Zeichenfolgevariablen als auch numerische Variablen verwendet werden. Datums- und Zeitvariablen sind jedoch nicht zulässig.

Dummy-Variable für Haupteffekte. Wählen Sie das Feld Dummys für Haupteffekte erstellen aus, um ein Set von Dummy-Variablen für die einzelnen ausgewählten Variablen zu erstellen.

Stammnamen. Geben Sie Namen an, die als Präfix für alle Dummy-Variablen für die einzelnen ausgewählten Variablen verwendet werden sollen. Geben Sie für jede ausgewählte Variable einen entsprechenden Namen an unde trennen Sie die einzelnen Namen durch ein Leerzeichen. Für die Dummy-Namen wird das Format "stammname_n" verwendet. Wenn eine Variable mit diesem Namen bereits vorhanden ist, wird sie ersetzt.

Makroname. Wenn Namen in dieses Feld eingegeben werden, wird für jedes Set von Dummy-Variablen oder metrischen Variablen ein Makro erstellt. Geben Sie für jede ausgewählte Variable einen entsprechenden Namen ein, wenn eine Makrogenerierung erforderlich ist. In der Regel wird ein Ausrufezeichen (!) als erstes Zeichen des Makronamens verwendet.

Zweiwege- und Dreiwegeinteraktionen. Gehen Sie hierbei genauso wie bei der Prozedur "Dummy-Variablen für Haupteffekte" vor. Geben Sie jedoch nur einen einzigen Stammnamen und einen Makronamen ein (bei Bedarf). Für Dummy-Namen wird das Format
"stammname_m_n" verwendet.
Vorhandene Variablen werden ersetzt.

Beschriftungen für Dummy-Variablen Beschriftungen für Dummy-Variablen werden standardmäßig aus dem Namen der Eingabevariablen und der entsprechenden Wertbeschriftung erstellt. Wählen Sie die Option Werte verwenden aus, um stattdessen die Werte zu verwenden. Wenn für einen Wert keine Beschriftung vorliegt, wird der Wert selbst in der Beschriftung verwendet.

Wertreihenfolge. Wählen Sie Aufsteigend oder Absteigend als Sortierreihenfolge der Dummy-Variablen aus. Dies ist vor allem in Verbindung mit der Erstellung eines Makros sinnvoll.

Erste Dummy-Kategorie in Makrodefinitionen übergehen. Wählen Sie dieses Feld aus, um die erste Dummy-Variable in den einzelnen Makrodefinitionen zu übergehen. Dies ist bei der Prozedur REGRESSION nützlich, um das Erstellen einer singulären Regressormatrix zu vermeiden. Wenn keine Interaktionsterme generiert werden, wird der erste Dummy für jede Variable in der Makrodefinition übergangen. Wenn keine Interaktionsterme vorhanden sind, werden Variablen, die diese Kategorie für eine der Eingabevariablen einschließen, in der Makrodefinition übergangen.

• Welche Kategorien als erste Kategorien betrachtet werden, ist von der Einstellung für "Wertreihenfolge" abhängig.

• Diese Einstellung wirkt sich nicht auf metrische Variablen aus.

Messniveauverwendung. Kategoriale Variablen erstellen einen Dummy für alle einzelnen Werte. Metrische Variablen werden durch eine einzige Variable dargestellt. Wählen Sie die Option Keine Dummys für metrische Variablenwerte erstellen für dieses Verhalten aus. Wählen Sie Dummys für alle Variablen erstellen aus, um das Messniveau zu ignorieren und eine Variable für die einzelnen Werte, einschließlich der metrischen Werte, zu erstellen.

Weitere Funktionen

In diesem Dialogfeld wird Syntax für den Erweiterungsbefehl SPSSINC CREATE DUMMIES generiert. Führen Sie folgende Syntax aus, um Hilfe für diesen Befehl anzuzeigen:

SPSSINC CREATE DUMMIES /HELP.

Anforderungen

Für dieses Dialogfeld sind das Integration Plug-In for Python und der Erweiterungsbefehl SPSSINC CREATE DUMMIES erforderlich. Bei IBM® SPSS® Statistics 19 und höher werden das Plug-In und der Erweiterungsbefehl mit dem Paket "Essentials for Python" installiert. Weitere Informationen finden Sie im Hilfesystem zu IBM SPSS Statistics unter "Core-System > Häufig gestellte Fragen" im Thema "Beziehen von Integrations-Plug-ins". Wenn Sie SPSSINC CREATE DUMMIES aus der SPSS Community heruntergeladen haben, folgen Sie den Anweisungen in der zugehörigen Readme-Datei.
