Crear un conjunto de variables auxiliares a partir de valores de variable

Este procedimiento acepta una lista de variables y produce un conjunto de variables auxiliares que representan sus valores. Puede producir valores auxiliares para variables individuales, valores auxiliares para términos de interacción bidireccional y/o valores auxiliares para interacciones tridireccionales. Pasa los datos omitiendo los casos excluidos por el filtro actual y crea valores auxiliares de los valores encontrados.

Para las variables de escala, de forma predeterminada, produce solo una variable en lugar de valores auxiliares que se pueden utilizar para producir los términos de interacción.

Crear variables auxiliares para. Elija las variables a partir de las que va a crear conjuntos de variables auxiliares. Habitualmente estas variables serían categóricas, pero puede resultar de utilidad seleccionar las variables de escala para compilar los términos de interacción. Se pueden usar variables de cadena y numéricas, pero no se permiten variables de fecha y tiempo.

Variables auxiliares de efecto principal. Seleccione la casilla Crear valores auxiliares de efecto principal para crear un conjunto de variables auxiliares para cada variable seleccionada.

Nombres raíz. Especifique los nombres que se van a utilizar como prefijo de todas las variables auxiliares para cada variable seleccionada. Especifique tantos nombres como variables seleccionadas, separadas por un espacio en blanco. Los nombres auxiliares tendrán el formato nombre-raíz_n. Si ya existe una variable con ese nombre, se sustituirá.

Nombre de macro. Si se especifican nombres en este campo, se creará una macro para cada conjunto de variables auxiliares o variable de escala. Especifique tantos nombres como variables seleccionadas si es necesaria la generación de macros. Es usual comenzar cada nombre de macro con "!".

Interacciones bidireccionales e interacciones tridireccionales. Siga el mismo procedimiento que para las variables auxiliares de efecto principal, pero especifique solo un único nombre raíz y nombre de macro (si lo desea). Los nombres auxiliares tendrán la forma
nombre-de-raíz_m_n,
y se sustituirán las variables existentes.

Etiquetas de variable auxiliar: de forma predeterminada, las etiquetas de las variables auxiliares se construyen a partir del nombre de variable de entrada y la etiqueta del valor adecuado. Seleccione Utilizar valores para utilizar en su lugar los valores. Si un valor no tiene etiqueta, se utiliza el mismo valor en la etiqueta.

Orden de valor. Seleccione Ascendente o Descendente para el orden de clasificación de las variables auxiliares. Es muy útil a la hora de generar una macro.

Omitir la primera categoría auxiliar de las definiciones de macro. Seleccione esta casilla de verificación para omitir la primera variable auxiliar de cada definición de macro. Resulta conveniente para el procedimiento REGRESSION para impedir que tenga una matriz de regresor singular. Si no hay términos de interacción generados, esto significa que el primer auxiliar de cada variable se omite de la definición de macro. Si hay términos de interacción, las variables relacionadas con esa categoría para las variables de entrada se omiten de la definición de macro.

• Depende del valor Orden de valor qué categorías se consideren primero.

• Las variables de escala no se ven afectadas por este valor.

Uso del nivel de medición. Las variables categóricas producen un valor auxiliar para cada valor distinto. Las variables de escala se representan mediante una única variable. Seleccione No crear valores auxiliares para los valores de variable de escala para obtener este comportamiento. Seleccione Crear valores auxiliares para todas las variables para omitir el nivel de medición y generar una variable para cada valor incluso para las variables de escala.

Características adicionales

Este cuadro de diálogo genera la sintaxis del comando de extensión SPSSINC CREATE DUMMIES. Para mostrar la ayuda de este comando, ejecute la siguiente sintaxis:

Requisitos

Este cuadro de diálogo requiere el complemento de integración de Python y el comando de extensión SPSSINC CREATE DUMMIES. Para IBM® SPSS® Statistics 19 y versiones posteriores, el complemento y el comando de extensión se instalan con el paquete Python Essentials. Si desea más información, consulte Cómo obtener complementos de integración, en Sistema principal > Preguntas más frecuentes en el sistema de ayuda de IBM SPSS Statistics. Si ha descargado SPSSINC CREATE DUMMIES de la Comunidad de SPSS, siga las instrucciones del archivo léeme asociado.
