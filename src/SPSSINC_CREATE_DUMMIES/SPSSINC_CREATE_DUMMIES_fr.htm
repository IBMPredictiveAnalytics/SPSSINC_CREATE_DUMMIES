Création d'un ensemble de variables dummy à partir des valeurs d'une variable

Cette procédure accepte une liste de variables et génère un ensemble de variables dummy représentant leurs valeurs. Elle permet de générer des variables dummy pour des variables individuelles, des termes d'interaction bidirectionnels et/ou des interactions tridirectionnelles. Elle transfère les données sans tenir compte des observations exclues par le filtre actuel et crée des variables dummy pour les valeurs trouvées.

Pour les variables d'échelle, par défaut, elle génère une seule variable au lieu de plusieurs variables dummy disponibles lors de la génération des termes d'interaction.

Créer des variables dummy pour : Choisissez les variables à partir desquelles vous souhaitez créer des ensembles de variables dummy. En règle générale, il s'agit de variables catégorielles, mais les variables d'échelle peuvent être utiles pour générer des termes d'interaction. Des variables de chaîne et numériques peuvent être utilisées mais pas des variables de date ou d'heure.

Effet principal des variables dummy : Cochez la case Créer l'effet principal des variables dummy pour créer un ensemble de variables dummy pour chaque variable sélectionnée.

Nom de racine : Indiquez le nom à utiliser comme préfixe de toutes les variables dummy pour chaque variable sélectionnée. Entrez autant de noms que de variables sélectionnées, en les séparant par un espace. Les noms factices auront la forme nomracine_n. Si une variable possède déjà ce nom, celui-ci sera remplacé.

Nom de macro : Si des noms sont saisis dans ce champ, une macro est créée pour chaque ensemble de variables dummy ou de variables d'échelle. Entrez autant de noms que de variables sélectionnées si la génération de macro est requise. Il est admis de commencer chaque nom de macro par "!".

Interactions bidirectionnelles et interactions tridirectionnelles : Suivez la même procédure que celle de l'effet principal des variables dummy, mais ne saisissez qu'un nom de racine et un nom de macro (si besoin). Les noms factices auront la forme
nomracine_m_n,
et les variables existantes seront remplacées.

Libellés de variables dummy : Par défaut, les libellés des variables dummy sont construits à partir du nom de la variable d'entrée et du libellé de valeur approprié. Sélectionnez Utiliser les valeurs pour utiliser les valeurs à la place. Si une valeur n'est pas étiquetée, la valeur elle-même est utilisée dans le libellé.

Ordre des valeurs : Choisissez l'ordre Croissant ou Décroissant pour les variables dummy. Ceci est utile lors de la génération de macros.

Ignorer la première catégorie de variable dummy dans les définitions de macro : Cochez cette case pour ignorer la première variable dummy dans la définition de chaque macro. Ceci se révèle particulièrement utile dans la procédure REGRESSION afin d'éviter d'avoir une matrice de facteurs de régression singulière. Si aucun terme d'interaction n'est généré, cela signifie que la première variable dummy de chaque variable est ignorée dans la définition de la macro. Si des termes d'interaction sont générés, les variables associées à cette catégorie pour n'importe quelle variables d'entrée sont ignorées de la définition de macro.

• Les catégories considérées comme premières varient en fonction du paramètre d'ordre des valeurs.

• Ce paramètre n'a aucune incidence sur les variables d'échelle.

Utilisation du niveau de mesure : Les variables catégorielles génèrent une variable dummy pour chaque valeur distincte. Les variables d'échelle sont représentées par une seule variable. Choisissez Ne pas créer de variables dummy pour les valeurs de variable d'échelle dans ce cas. Choisissez Créer des variables dummy pour toutes les variables pour ne pas tenir compte du niveau de mesure et générer une variable pour chaque valeur, y compris pour les variables d'échelle.

Fonctions supplémentaires

Cette boîte de dialogue génère une syntaxe pour la commande d'extension SPSSINC CREATE DUMMIES. Pour afficher l'aide de cette commande, exécutez la syntaxe suivante :

SPSSINC CREATE DUMMIES /HELP.

Conditions requises

Cette boîte de dialogue requiert le plug-in d'intégration pour Python et la commande d'extension SPSSINC CREATE DUMMIES. Pour IBM® SPSS® Statistics 19 et versions ultérieures, le plug-in et la commande d'extension sont installés avec le pack Essentials for Python. Pour plus d'informations, reportez-vous à Comment obtenir des plug-in d'intégration dans Système central > Questions fréquemment posées dans le système d'aide d'IBM SPSS Statistics. Si vous avez téléchargé SPSSINC CREATE DUMMIES depuis la communauté SPSS, suivez les instructions dans le fichier associé Lisezmoi.
