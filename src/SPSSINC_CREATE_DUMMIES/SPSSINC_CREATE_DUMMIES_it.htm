Crea un insieme di variabili dummy dai valori di una variabile

Questa procedura accetta un elenco di variabili e produce un insieme di variabili dummy che rappresentano i loro valori. Può produrre valori dummy per singole variabili, per termini di interazione a due vie e/o per interazioni a tre vie. Passa i dati ignorando i casi esclusi dal filtro corrente e crea valori dummy per i valori trovati.

Per le variabili di scala, per impostazione predefinita, produce solo una variabile invece dei valori dummy che è possibile utilizzare nella produzione di termini di interazione.

Crea variabili dummy per. Scegliere le variabili da cui creare insiemi di variabili dummy. In genere si tratta di variabili categoriali, ma è possibile selezionare anche le variabili di scala per creare i termini di interazione. È possibile utilizzare sia le variabili stringa che numeriche, ma le variabili data e tempo non sono consentite.

Variabili dummy effetto principale. Selezionare la casella Crea valori dummy effetto principale per creare un insieme di variabili dummy per ogni variabile selezionata.

Nomi radice. Specificare i nomi da utilizzare come prefisso di tutte le variabili dummy per ogni variabile selezionata. Immettere un numero di nomi pari al numero di variabili selezionate separati da uno spazio. I nomi dummy avranno il formato rootname_n. Se già esiste una variabile con quel nome, viene sostituita.

Nome di macro. Se si immettono dei nomi in questo campo, verrà creata una macro per ogni insieme di variabili dummy o variabili di scala. Immettere un numero di nomi pari al numero di variabili selezionate se è richiesta la generazione di macro. Per convenzione, ogni nome di macro inizia con "!".

Interazioni a due vie e a tre vie. Seguire la stessa procedura seguita per Variabili dummy effetto principale, ma immettere un solo nome radice e nome di macro (se lo si desidera). I nomi dummy hanno il formato
rootname_m_n,
e le variabili esistenti verranno sostituite.

Etichette di variabili dummy. Per impostazione predefinita, le etichette per le variabili dummy vengono create dal nome della variabile di input e dall'etichetta valore appropriata. Selezionare Utilizza valori per utilizzare i valori. Se un valore è senza etichetta, verrà utilizzato nell'etichetta.

Ordine valori. Scegliere Crescente o Decrescente per il criterio di ordinamento delle variabili dummy. Ciò è particolarmente utile nella connessione con la creazione di una macro.

Ometti prima categoria dummy dalle definizioni di macro. Selezionare questa casella per omettere la prima variabile dummy da ogni definizione di macro. Ciò è utile per la procedura REGRESSION per evitare di avere una sola matrice di regressione. Se non vengono generati dei termini di interazione, il primo valore dummy per ogni variabile viene omesso dalla definizione della macro. Se esistono dei termini di interazione, le variabili che implicano quella categoria per una qualsiasi delle variabili di input vengono omesse dalla definizione della macro.

• Quali categorie vengono considerate per prime dipende dall'impostazione Ordine valori.

• Le variabili di scala non vengono interessate da questa impostazioni.

Utilizzo del livello di misurazione. Le variabili categoriali producono un valore dummy per ogni singolo valore. Le variabili di scala sono rappresentate da una singola variabile. Scegliere Non creare valori dummy per i valori delle variabili di scala per ottenere questo comportamento. Scegliere Crea valori dummy per tutte le variabili per ignorare il livello di misurazione e generare una variabile per ogni valore, anche per le variabili di scala.

Funzioni aggiuntive

Questa finestra di dialogo genera la sintassi per il comando di estensione SPSSINC CREATE DUMMIES. Per visualizzare la guida per questo comando, utilizzare la sintassi riportata di seguito:

SPSSINC CREATE DUMMIES /HELP.

Requisiti

Questa finestra di dialogo richiede Integration Plug-In for Python e il comando di estensione SPSSINC CREATE DUMMIES. Per IBM® SPSS® Statistics 19 e versioni successive, il plug-in e il comando di estensione vengono installati con il package Essentials for Python. Per ulteriori informazioni, consultare Come ottenere i plug-in di integrazione in Sistema principale > Domande frequenti nella guida in linea di IBM SPSS Statistics. Se il comando di estensione SPSSINC CREATE DUMMIES è stato scaricato da SPSS Community, seguire le istruzioni nel file readme associato.
