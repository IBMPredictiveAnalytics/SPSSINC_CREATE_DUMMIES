Criar um Conjunto de Variáveis Dummy a Partir de Valores de uma Variável

Este procedimento aceita uma lista de variáveis​e produz um conjunto de variáveis dummy representando seus valores. Ele pode produzir variáveis dummy para variáveis individuais, variáveis dummy para termos de interação de dois fatores e/ou interação de três fatores. Ele passa os dados ignorando quaisquer casos excluídos pelo filtro atual e cria variáveis dummy para os valores localizados.

Para variáveis escalares, por padrão, ele produz apenas uma variável ao invés de variáveis dummy que podem ser usadas na produção de termos de interação.

Criar Variáveis Dummy para. Escolha variáveis a partir das quais criar conjuntos de variáveis dummy. Normalmente, essas seriam as variáveis categóricas, mas variáveis de escala podem ser selecionadas de maneira útil para a construção de termos de interação. Tanto as variáveis de sequência de caracteres e numérica podem ser usadas, mas variáveis de data e hora não são permitidas.

Variáveis Dummy de Efeito Principal. Marque a caixa Criar Simulados de Efeito Principal para criar um conjunto de variáveis dummy para cada variável selecionada.

Nomes Raiz. Especifique nomes a serem usados como o prefixo de todas as variáveis dummy para cada variável selecionada. Insira quantos nomes houver que sejam variáveis selecionadas, separados por um espaço em branco. Os nomes dummy terão o formato rootname_n. Se uma variável com tal nome já existir, ela será substituída.

Nome da Macro. Se os nomes são inseridos neste campo, uma macro será criada para cada conjunto de variáveis dummy ou variáveis de escala. Insira quantos nomes houver que sejam variáveis selecionadas se a geração da macro for requerida. O convencional é iniciar cada nome de macro com "!".

Interações de Dois Fatores e Interações de Três Fatores. Siga o mesmo procedimento para Variáveis Dummy de efeito principal, mas insira apenas um nome raiz único e um nome de macro (se desejado). Nomes dummy terá o formato
rootname_m_n,
e as variáveis existentes serão substituídas.

Rótulos de Variáveis Dummy Por padrão, os rótulos para as variáveis dummy são construídos a partir do nome de variável de entrada e o rótulo de valor apropriado. Selecione Usar valores para usar os valores no lugar. Se um valor não for rotulado, o próprio valor será usado no rótulo.

Ordem dos Valores. Escolha Crescente ou Decrescente para a ordenação das variáveis dummy. Isso é principalmente útil em conjunto com a geração de uma macro.

Omitir primeira categoria dummy das definições da macro. Marque esta caixa para omitir a primeira variável dummy a partir de cada definição de macro. Isto é conveniente para o procedimento REGRESSION para evitar a necessidade de ter uma matriz regressora singular. Se não houver termos de interação que estão sendo gerados, isso significa que o primeiro dummy para cada variável é omitido da definição da macro. Se houver termos de interação, as variáveis que envolvem essa categoria para quaisquer variáveis de entrada são omitidas da definição da macro.

• Quais categorias são consideradas como as primeiras dependem da configuração de Ordem de Valores.

• Variáveis de escala não são afetadas por esta configuração.

Uso de Nível de Medição. As variáveis categóricas produzem um dummy para cada um dos valores distintos. Variáveis de escala são representadas por uma variável única. Escolha Não criar variáveis dummy para valores de variável escalares para este comportamento. Escolha Criar variáveis dummy para todas as variáveis para ignorar o nível de medição e gerar uma variável para cada valor, mesmo para variáveis escalares.

Recursos adicionais

Esta caixa de diálogo gera sintaxe para o comando de extensão SPSSINC CREATE DUMMIES. Para exibir ajuda para este comando, execute a seguinte sintaxe:

SPSSINC CREATE DUMMIES /HELP.

Requisitos

Esta caixa de diálogo exige o plug-in de integração para Python e o comando de extensão SPSSINC CREATE DUMMIES. Para o IBM® SPSS® Statistics 19 e posterior, o Plug-In e o comando de extensão são instalados com o pacote Essentials for Python. Para obter informações adicionais, consulte Como Obter Integration Plug-In, no Sistema Principal > Perguntas mais Frequentes no sistema de ajuda do IBM SPSS Statistics Help. Se você tiver feito o download de SPSSINC CREATE DUMMIES da Comunidade SPSS, siga as instruções do arquivo Leia-me associado.
